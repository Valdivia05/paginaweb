Protocolos http

Es un protoclo de comunicacion, permite la transferencia de informacion atravez de archivos en la w.w.w

Query string

Cadena de consulta, se utiliza para hacer referencia a una interaccion con una base de datos

Verbos http

DNS

Sistema que ayuda a traducir los dominios a direccion ip para la lectura humana

CDN

Red de servidores interconectadas, ayuda a la eficienda de cargas de paginas con exesivo uso de datos
